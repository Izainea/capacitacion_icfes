Programa de Formación

Desarrollo de Tableros de Resultados Avanzados con R-Shiny para el ICFES

Contenido Académico

Bloque I: Fundamentos y Prototipado

MÓDULO 1: Principios y Ecosistema

Temas: Ciclo de vida MLOps. R-Shiny vs. Power BI. Metodologías de proyectos de datos. Configuración del entorno con Git/GitHub.

MÓDULO 2: Análisis Exploratorio Interactivo (EDA)

Temas: Creación de una primera app para EDA. `ggplot2` y `plotly`. Diseño de layouts básicos.

MÓDULO 3: Estructura y Reactividad

Temas: El paradigma de reactividad. Optimización con `reactive()`. Aplicaciones modulares (`ui`/`server`).

Bloque II: Desarrollo del Producto de Datos

MÓDULO 4: Pipelines de Datos en Shiny

Temas: Carga y transformación de datos (`dplyr`). Implementación de pipelines de inferencia. Manejo de errores.

MÓDULO 5: Integración de Modelos

Temas: Desacoplamiento de artefactos. Carga de modelos pre-entrenados (`.rds`). Creación de simuladores.

MÓDULO 6: Visualización Avanzada

Temas: Tablas interactivas con `DT`. Mapas con `leaflet`. Interfaces complejas.

MÓDULO 7: Generación de Artefactos

Temas: Reportes parametrizados con `rmarkdown`. Implementación de descargas. Automatización.

Bloque III: Profesionalización y Operaciones

MÓDULO 8: Arquitectura Profesional

Temas: Diseño con `shinydashboard`. Branding y estética institucional.

MÓDULO 9: Conectividad y Gobernanza

Temas: Conexión a bases de datos y APIs. Gestión de credenciales y seguridad.

MÓDULO 10: Control de Versiones Avanzado

Temas: Flujos de trabajo colaborativos con Git. Trazabilidad de cambios.

MÓDULO 11: Despliegue y Mantenimiento (CI/CD)

Temas: Gestión de dependencias (`renv`). Despliegue. Introducción a Docker y CI/CD.

MÓDULO 12: Proyecto Final

Temas: Desarrollo guiado de un producto de datos completo. Simulación de un ciclo MLOps.
