Módulo 6: Visualización y Storytelling Interactivo

De la Exploración al Storytelling: Comunicando Insights con Claridad y Precisión.

Agenda del Módulo

Lección 6.1

Fundamentos: Gráficos con Propósito

Repaso de `ggplot2`. Aprende a diferenciar entre visualizaciones para explorar (para el analista) y para explicar (para la audiencia), sentando las bases del Data Storytelling.

Iniciar Lección
Lección 6.2

Deep Dive: Gráficos Interactivos con `plotly`

Transforma tus gráficos estáticos en experiencias dinámicas con `ggplotly()`. Aprende a personalizar tooltips para revelar detalles y a explorar eventos para una interactividad avanzada.

Iniciar Lección
Lección 6.3

Masterclass: Data Storytelling con Datos ICFES

Aprende a contar una historia. Aplica principios clave como el uso estratégico del color, anotaciones y el enfoque para guiar al usuario hacia los insights más importantes de los puntajes.

Iniciar Lección
Lección 6.4

Deep Dive: Tablas Interactivas con `DT`

Usa tablas como una lupa para el análisis detallado. Implementa filtros, paginación, y opciones de exportación a hojas de cálculo para complementar tus visualizaciones y empoderar al usuario.

Iniciar Lección

Laboratorio Interactivo

Complementando las lecciones, hemos creado una aplicación Shiny que funciona como una galería interactiva. Ejecuta el archivo `app.R` en la raíz del proyecto para explorar más de 30 visualizaciones, ver su código y experimentar con ellas en tiempo real.
