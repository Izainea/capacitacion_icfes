Módulo 8: Arquitectura Profesional

De un script funcional a un producto de datos robusto, intuitivo y con identidad institucional.

Contenido del Módulo

La Estructura de `shinydashboard`

Aprende a construir el esqueleto de una aplicación profesional con Header, Sidebar y Body, sentando las bases de un layout limpio y navegable.

Branding y Estética Institucional

Integra la identidad visual del ICFES en tu dashboard. Personaliza colores, tipografías y estilos usando CSS para crear una experiencia de usuario cohesiva.

Layouts Avanzados y Buenas Prácticas

Domina componentes avanzados como `valueBox` y `tabBox` para presentar KPIs de alto impacto y organizar el contenido de manera eficiente.

Diseño Asistido por IA

Actividad Práctica: Usa un LLM como co-piloto estratégico para definir KPIs, estructurar la narrativa y auditar la UX antes de escribir una línea de código.
